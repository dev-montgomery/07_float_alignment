About Using Floats

Floats have become somewhat antiquated with the introduction of flexbox and grid. The point of this assignment is just to learn about floats.

More About Floats

When you use floats, you'll have to consider how css such as padding affects using values like percentages. To get around that complication, you can add a box-sizing attribute of border-box to the body or to all using an asterisk.

Another Thing About Using Floats

When using floats, you have to consider how elements can consequently overlap. You end up needing to clear your floats using an element in the right placement on your page styled with a "clear: both" attribute.

Last Thing About Floats

They suck. Maybe that's an oversimplified statement, but honestly, they suck. Responsive design is critical, and floats are just a very inelegant solution.
